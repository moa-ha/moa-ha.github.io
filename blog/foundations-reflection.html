Foundations-Reflection

Three new things I have learnt about myself due to the core learning.

Firstly, I hesitate to ask but wanted to finish on my own if possible. As a learner, it is kind of needed to get something completely, but asking for help will effective in many way so will need to be used to it. Secondly, I have learnt that I value performance or result more than others. It means creativity or kindness was behind which is a key value for teamwork and I will need to put more effort on this. Lastly, I tend to set the goal I could achieve and it might limit my potential.

In learning and programming,

  • VALUE affects learning style and the way of setting goal.
  • EMPATHY is needed for teamwork and most of programming is team project.
  • SELF-AWARENESS makes to be able to start over not to give up even though something's gone a little wrong.

What surprised me the most about the core learning and the most challenging aspects.

The fact that learning ability can be improved by mindset is surprising. Because I’ve been planning what I could do realistically, there was not much challenge, but setting a higher goal can raise my ability.
However, actually changing mindset in real that I could do more is not easy.

I believe DEV is spending much time focusing on CORE LEARNING because it will lead me not only to successful performance as a developer but also to be a good teammate.
I needed this rather than just programming course to set a long-term goal taking a new turn, and I got to know where I should put more effort on, so it was more than worth it.
